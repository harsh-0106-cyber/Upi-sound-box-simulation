🎓 Presentation Flow

1. Open each interface on a separate device  →  2. Send a legitimate payment from Sender  →  3. Sound Box announces it  →  4. Switch to Attacker & send a fake payment with Security OFF (it succeeds!)  →  5. Toggle Security ON on the Sound Box  →  6. Repeat the attack (it gets blocked with detailed logs)
